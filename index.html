RentEase Platform

Rôle
Développeur Full-Stack
Date
Octobre - Décembre 2022
Client
Bzcorp

RentEase est une solution logicielle complète conçue pour simplifier la location et la gestion immobilière. Elle permet aux propriétaires de gérer leurs biens de manière proactive et offre aux locataires un espace moderne pour le suivi de leurs contrats et paiements.

Fonctionnalités clés

  • Tableau de bord dynamique pour l'analyse des revenus et le taux d'occupation des propriétés.
  • Gestion de contrats de bail dématérialisés avec signature électronique fictive.
  • Système de tickets de réclamations avec notification en temps réel pour le traitement des pannes de maintenance.
  • Base de données relationnelle hautement normalisée conçue sous PostgreSQL et gérée par Hibernate.

Technologies employées

Backend : Java 17, Spring Boot, Spring Security (JWT), Hibernate, JPA.
Frontend : Angular 14, RxJS, TypeScript, TailwindCSS.
Base de données : PostgreSQL, MySQL.
